## Semigroup Solution

### Phase Space Construction

The solution space depends on boundary conditions (1.6). For Dirichlet conditions (a):

```
H = H₀¹(0,l) × L²(0,l) × H₀¹(0,l) × L²(0,l) × L²(0,l) × L²(0,l)
```

For mixed conditions, we use:
- H₀¹(0,l): functions vanishing at endpoints
- H₁∗(0,l) = H¹(0,l) ∩ L²∗(0,l) where L²∗(0,l) = {u ∈ L²(0,l) | ∫₀ˡ u(x)dx = 0}

**Hilbert space norm:**

```
||U||²H = ∫₀ˡ [ρ₁|Φ|² + ρ₂|Ψ|² + b|ψx|² + k|ϕx + ψ|² + ρ₃|θ|² + ρ₄|ϑ|²] dx
```

for U = (ϕ, Φ, ψ, Ψ, θ, ϑ) ∈ H, where Φ = ϕt and Ψ = ψt are velocity components.

### Abstract First-Order Formulation

System (1.1)-(1.6) is rewritten as:

```
Ut = AU, t > 0
U(0) = U₀ = (ϕ₀, ϕ₁, ψ₀, ψ₁, θ₀, ϑ₀)
```

The operator A : D(A) ⊂ H → H is given by:

```
AU = (Φ, k(ϕx+ψ)x/ρ₁ − mθx/ρ₁, Ψ, bψxx/ρ₂ − k(ϕx+ψ)/ρ₂ + mθ/ρ₂ − σϑx/ρ₂, 
      c₀θxx/ρ₃ − m(Φx+Ψ)/ρ₃, c₁ϑxx/ρ₄ − σΨx/ρ₄)ᵀ
```

### Domain of the Operator

```
D(A) = {U ∈ H | ϕ, ψ, θ, ϑ ∈ H²(0,l) and (D) is satisfied}
```

where (D) consists of the boundary condition constraints for each of the eight boundary condition cases (a)-(h).

For Dirichlet case (a):

```
D(A) = {U ∈ H | ϕ, ψ, θ, ϑ ∈ H²(0,l), θ, ϑ, Φ, Ψ ∈ H₀¹(0,l)}
```

### Dissipativity

A fundamental property is that the operator A is dissipative:

```
Re(AU, U)H = −c₀∫₀ˡ|θx|² dx − c₁∫₀ˡ|ϑx|² dx ≤ 0
```

This holds **independent of boundary conditions**, confirming full dissipation through both thermal variables.

## Semigroup Generation and Well-Posedness

### Theorem 3.1: Existence and Uniqueness

**Statement**: Let A : D(A) ⊂ H → H be given. Then:

1. If U₀ ∈ H, then problem has a unique mild solution U ∈ C⁰([0,∞), H)

2. If U₀ ∈ D(A), then the mild solution is regular: U ∈ C⁰([0,∞), D(A)) ∩ C¹([0,∞), H)

3. If U₀ ∈ D(Aⁿ), n ≥ 2 integer: U ∈ ∩ⁿⱼ₌₀ Cⁿ⁻ʲ([0,∞), D(Aʲ))

**Proof Strategy**: By semigroup theory (Pazy), it suffices to show:
- A is dissipative (proven above)
- (Id − A) : D(A) → H is onto (surjective)

The surjectivity is demonstrated using the Lax-Milgram theorem on an appropriate variational formulation. For any F = (f₁, f₂, f₃, f₄, f₅, f₆) ∈ H, we solve U − AU = F through:

```
∫₀ˡ [ρ₁ϕϕ̃ + ρ₂ψψ̃ + ρ₃θθ̃ + ρ₄ϑϑ̃] dx + ∫₀ˡ [k(ϕx+ψ)(ϕ̃x+ψ̃) + bψxψ̃x 
+ c₀θxθ̃x + c₁ϑxϑ̃x] dx − ∫₀ˡ [mθ(ϕ̃x+ψ̃) − σϑψ̃x] dx 
+ ∫₀ˡ [m(ϕx+ψ)θ̃ + σψxϑ̃] dx = ∫₀ˡ (ρ₁f₁+ρ₁f₂)ϕ̃ + (ρ₂f₃+ρ₂f₄)ψ̃ 
+ (mf₁,x + mf₃ + ρ₃f₅)θ̃ + (σf₃,x + ρ₄f₆)ϑ̃ dx
```

for all test functions (ϕ̃, ψ̃, θ̃, ϑ̃) in appropriate spaces.

## Physical Interpretation

The well-posedness result means:

1. **Existence**: For any reasonable initial data, a unique solution exists

2. **Regularity**: Solutions become smooth (twice differentiable in space, once in time) instantaneously if initial data is in the domain

3. **Continuous dependence**: Small changes in initial data lead to small changes in solutions

## Semigroup of Contractions

The semigroup T(t) = eᴬᵗ generated by A satisfies:

```
||T(t)U₀||H ≤ ||U₀||H, t ≥ 0
```

This contraction property ensures energy dissipation: the total energy of the system never increases in time.

## Basis for Stability Analysis

Well-posedness establishes that:
- Solutions exist for all time
- The system generates a C₀-semigroup of contractions
- Domain of A is densely embedded and compactly embedded in H

These properties provide the foundation for proving exponential stability in Section 4 using Gearhart-Huang-Prüss spectral theory.
