# Nomenclature / Definition of Terms

| Symbol | Definition |
|--------|-----------|
| A | Cross-sectional area of inner cylinder |
| E | Young's modulus of isotropic material; effective axial modulus of laminated composite cylinder |
| g | Acceleration due to gravity |
| h | Thickness of inner cylinder (in thin-walled buckling formulas) |
| I | Moment of inertia of inner cylinder cross-section |
| L | Length of inner cylinder |
| m | Number of times (full wavelengths) helical buckle travels around circumference of constraining outer cylinder |
| n | Number of half-wavelengths of sinusoidal buckle |
| p | Pitch (wavelength) of sinusoidal or helical buckle |
| P | Applied axial load |
| Pʰ | Axial load for post-buckled constrained Euler helically buckled cylinder |
| Pʰ_cr | Axial buckling load for constrained Euler helical buckle |
| Pˢ_cr | Axial buckling load for constrained Euler sinusoidal buckle |
| R | Nominal average radius of inner cylinder in thin-walled buckling formulas |
| T | Applied twisting torque |
| uᵢ | Displacement of the centerline of the inner cylinder in the i-th direction |
| Wₙ | Normal force per unit length exerted on inner cylinder by outer cylinder |
| w | Weight per unit length of inner cylinder (ρAg) |
| x | Element of right-handed Cartesian frame, axis of inner cylinder |
| y | Element of right-handed Cartesian frame, direction of gravity |
| z | Element of right-handed Cartesian frame |
| β | Angle from vertical of buckled cylinder |
| δ | Maximum vertical deflection of cylinder under its own weight |
| Δr | Radial clearance between inner and outer cylinders |
| φ | Empirical knockdown factor for concertina thin-walled buckling mode |
| ρ | Density of inner cylinder material |
| μ | Coefficient of friction between inner and outer cylinders |
| ν | Poisson's ratio of isotropic cylinder |
