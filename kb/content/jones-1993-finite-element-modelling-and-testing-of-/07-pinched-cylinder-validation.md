## Pinched Orthotropic Cylinder: Validation Study

### Choice of Test Case

A pinched cylinder (diametral point load on a cylindrical shell) was chosen as the validation case because:
- It is a well-known benchmark problem in shell mechanics
- The geometry is simple enough to manufacture accurately but complex enough to challenge the FE model
- An analytical solution exists (Calladine) that can be extended to orthotropic materials

The specimen is a filament-wound GRP tube with:
- Varying winding angle over the surface (representative of a real component)
- Alternating lamination sequence (captured by FILFEM I)
- Measurement of diametral deflection per unit load

### Analytical Solution (Calladine Extended)

Calladine's solution for an isotropic pinched cylinder introduces the parameter:

```
c = [3(1 - v^2)]^(1/4) * (a/t)^(1/2)
```

where a = radius and t = thickness. The author extended this solution to orthotropic materials by replacing isotropic properties with orthotropic equivalents, using classical lamination theory to derive the effective meridional and circumferential flexural rigidities.

### Finite Element Models

Several FE models were generated by FILFEM I and tested:

| Model | Element type | System |
|-------|-------------|--------|
| A1 | S4R thin shell | ABAQUS |
| A2 | S8R thin shell | ABAQUS |
| P1 | Semi-Loof thin shell | PAFEC |
| P2 | Ahmad thick shell | PAFEC |
| P3 | Axisymmetric Fourier | PAFEC |

### Results

- FE results using best-fitted unidirectional material properties showed excellent agreement with experimental diametral deflections
- The extended Calladine analytical solution agreed closely with the most comparable FE model
- Results with nominal (Halpin-Tsai) material properties showed larger discrepancies, reflecting the scatter in material property measurements

### Conclusions

The validation demonstrates that:
1. FILFEM I successfully generates accurate FE models from CADFIL I data
2. Material property data is the critical factor in obtaining good agreement with experiment
3. The back-fitting approach to unidirectional property determination gives excellent stiffness predictions
4. The extended Calladine analytical solution provides a useful independent check on FE results

### Practical Implication

For future applications of FILFEM I, confidence in the FE results depends on obtaining reliable material properties through carefully controlled tests. The inconsistency observed with glass/polyester specimens suggests that glass/epoxy systems would provide more repeatable results for model validation.
